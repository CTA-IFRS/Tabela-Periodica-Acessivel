Dicas

Para utilizar a tabela periódica com o Jaws utilize as seguintes configurações:

  • Ative o menu de configurações com o atalho Insert + V.
  • Procure pela opção "Link apenas de texto" e marque-o para "título".
  • Procure pela opção "Título das tabelas" e marque-o para "desactivado".
  • Para navegar utilize os atalhos Ctrl + Alt + seta direcional.
